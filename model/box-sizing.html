box-sizing

css

No modelo "box-model" do CSS o tamanho final que um elemento ocupa de espaço de template será a sua largura mais o espaço de padding e border.

A propriedade box-sizing permite alterar esse calculo para considerar apenas o tamanho do conteúdo do elemento ao incorporar o espaço do padding e border no conteúdo do elemento.

padrão: content-box

width: 300px; box-sizing: content-box width real:

com border-box

width: 300px; box-sizing: border-box width real:
