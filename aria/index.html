html

Permite a marcação de regiões importantes na página (como uma caixa de busca, um cabeçalho, chamadas "pontos de referência") para facilitar a navegação em leitores de tela

Add machine-readable semantics for an assistive technology to html elements.

Enables developers to describe their widgets in more detail by adding special attributes to the markup.

Provides roles and states that describe the behavior of most familiar UI widgets.

ROLE

Roles describe widgets that aren't otherwise available in HTML 4, such as sliders, menu bars, tabs, and dialogs.

An element with role="slider" will be controlled in the same way as a native slider is controlled on the operating system.

Rolde can be used, for example, by applications and assistive technologies to determine the purpose of an element.
